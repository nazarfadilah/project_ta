Kebutuhan Fungsional baru berdasarkan perubahan hak akses yang telah kita buat:
1) Users dapat melakukan login dan logout
2) Tamu dapat melakukan registrasi akun
3) Admin dapat mengelola data Users
4) Admin dapat mengelola data tentang/landing page
5) Admin dan Petugas dapat mengelola data tamu
6. Tamu dapat mengelola data ulasan
7) Petugas dapat mengelola data ruangan
8) Petugas dapat mengelola data sarana
9) Petugas dapat mengelola data paket ruangan
10) Tamu dapat mengelola profil
11) Petugas dapat mengelola data invoice
12) Petugas dapat melakukan check-in peminjaman
13) Petugas dapat melakukan check-out peminjaman
14) Petugas dapat melakukan verifikasi peminjaman
15) Peminjam/tamu dapat melakukan permohonan
16) Peminjam/tamu dapat melakukan pembatalan
17) Admin dan Pimpinan dapat melihat laporan
18) Admin dan Petugas dapat mengelola berita
19) Admin dapat mempublish berita
